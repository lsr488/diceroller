Challenge #364: Create a Dice Roller

posted by u/jnazario

Input Description

Create a set of inputs that allow you to roll any number of dice, with any number of faces. For example:

Your input will contain one or more lines, where each line will be in the form of "NdM"; for example:

3d6

4d12

1d10

5d4

The first number is the number of dice to roll (the d means "dice"). This can be any integer between 1 and 100, inclusive.

The second number is how many sides each die has. This can be any integer between 2 and 100, inclusive.

The above example of "3d6" means "3 6-sided dice".

Output Description

Output the sum of each set of rolled dice, on their own lines. For example, if you rolled 3d6:

14

Just a single number, you rolled 3 6-sided dice, and they added up to 14.

Notes/Hints

Don't just multiply your roll by the number of dice. Roll each die individually.

Bonus

In addition to the sum of all dice rolls for your output, print out the result of each roll on the same line, using a format that looks something like:

14: 6 3 5

22: 10 7 1 4

9: 9

11: 3 2 2 1 3
